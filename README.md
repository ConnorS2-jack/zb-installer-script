zb-installer-script
installs zen browser with a sh file in most distros! (as root)

```bash
curl -fsSL https://raw.githubusercontent.com/MalikHw/zb-installer-script/main/install-zen.sh | bash
```
for forced yes:
```
curl -fsSL https://raw.githubusercontent.com/MalikHw/zb-installer-script/main/install-zen.sh | bash -s -- --yes
```
